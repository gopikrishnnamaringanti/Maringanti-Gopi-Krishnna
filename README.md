This website is built using the key concepts of Html, Css and JavaScript.

The functionalities of this website are:
It will act as a model for drag and drop of items from one container from another container within the page.

The first container will be initially loaded with six elements with some files, images and text.

We can drag the files from one container to the other by using cursor and the UI will be updated as the container2 will be updated visually with the files from the 
container1.

There will be a reset button by pressing it the containers will be reset and restored to their original conditions.
